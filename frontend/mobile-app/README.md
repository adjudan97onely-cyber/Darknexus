# Darknexus - Version mobile sans PC

## Objectif
Avoir Darknexus sur telephone comme une vraie application, sans laisser le PC allume.

## Solution recommandee
1. Deployer Darknexus frontend sur Vercel.
2. Ouvrir l'URL sur mobile.
3. Installer la PWA depuis le navigateur (Ajouter a l'ecran d'accueil).

## Option Android APK
1. Copier l'URL de production.
2. Utiliser https://www.pwabuilder.com/
3. Generer puis installer l'APK.

## Important
Un ZIP GitHub n'est pas une app mobile installable.
Pour une vraie app: PWA installee ou APK.
