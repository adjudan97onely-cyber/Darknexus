## 🛠️ EN CAS DE PROBLÈME

### ❌ "Python n'est pas reconnu..."
➡️ Python n'est pas installé OU la case "Add to PATH" n'a pas été cochée
**Solution** : Réinstalle Python en cochant bien "Add Python to PATH"

### ❌ "Node n'est pas reconnu..."
➡️ Node.js n'est pas installé
**Solution** : Installe Node.js depuis https://nodejs.org/

### ❌ Le backend ne démarre pas
➡️ MongoDB n'est pas lancé
**Solution** : Lance MongoDB Compass ou `mongod.exe`

### ❌ L'installation plante
➡️ Problème de connexion internet pendant l'installation des dépendances
**Solution** : 
1. Vérifie ta connexion internet
2. Relance `INSTALLATION_AUTOMATIQUE.bat`

### ❌ Je ne trouve pas le fichier `.bat`
➡️ Tu cherches au mauvais endroit
**Solution** : 
1. Va dans le dossier où tu as **extrait** le code (pas le fichier .zip !)
2. Cherche `INSTALLATION_AUTOMATIQUE.bat`
