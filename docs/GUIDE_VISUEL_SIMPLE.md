## ❓ EN CAS DE PROBLÈME

### ❌ "Python n'est pas reconnu..."

```
🔴 PROBLÈME : Python pas installé ou pas dans PATH

✅ SOLUTION :
1. Va sur : https://www.python.org/downloads/
2. Télécharge Python
3. ⚠️  COCHE LA CASE "Add Python to PATH" !!!
4. Installe
5. Redémarre ton PC
6. Relance INSTALLATION_AUTOMATIQUE.bat
```

### ❌ Le backend ne démarre pas

```
🔴 PROBLÈME : MongoDB n'est pas lancé

✅ SOLUTION :
1. Ouvre MongoDB Compass
2. OU ouvre CMD et tape : mongod
3. Relance LANCER_DARK_NEXUS.bat
```

### ❌ Je ne trouve pas les fichiers .bat

```
🔴 PROBLÈME : Tu es dans le mauvais dossier

✅ SOLUTION :
1. Assure-toi d'avoir EXTRAIT le fichier .zip
2. Va dans le dossier EXTRAIT (pas le .zip !)
3. Cherche les fichiers :
   - INSTALLATION_AUTOMATIQUE.bat
   - LANCER_DARK_NEXUS.bat
```
