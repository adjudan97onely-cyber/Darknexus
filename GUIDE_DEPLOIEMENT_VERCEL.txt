RÈGLES IMPORTANTES QUE TU DOIS RESPECTER :

1. PWA — ICÔNES :
   La règle SPA "/(.*) → index.html" bloque les images si elles sont
   dans /assets/images/. Les icônes du manifest.json DOIVENT être
   dans frontend/public/ (ex: icon-192.png, icon-512.png) pour être
   accessibles directement.

2. PWA — FICHIERS REQUIS :
   - public/manifest.json (nom, icônes, couleurs, display: standalone)
   - public/sw.js (service worker avec install + fetch handlers)
   - public/icon-192.png et icon-512.png
   - Dans app/+html.tsx : <link rel="manifest" href="/manifest.json" />
     et le script d'enregistrement du SW

3. DÉPLOIEMENT :
   - Je pousse sur GitHub (branche main)
   - Vercel redéploie automatiquement (~2 min)
   - URL format : nom-projet.vercel.app

4. POUSSER LES CHANGEMENTS :
   Je clone localement, tu fais les modifications, et je pousse avec :
   git push origin main
