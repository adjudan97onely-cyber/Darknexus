# GUIDE D'INSTALLATION WINDOWS - DARK NEXUS AI 2042

## PROBLEME AVEC L'ANCIEN SCRIPT

Le premier script contenait des emojis et caracteres speciaux qui ne fonctionnent pas sur Windows CMD.

Le NOUVEAU script "INSTALLATION_AUTOMATIQUE_v2.bat" est PROPRE et fonctionne parfaitement !

---

## ETAPE 1 : PRE-REQUIS (Installe ces logiciels AVANT)

### Python
- Telechargement : https://www.python.org/downloads/
- IMPORTANT : Coche "Add Python to PATH" pendant l'installation !

### Node.js
- Telechargement : https://nodejs.org/ (prends la version LTS)

### MongoDB
- Telechargement : https://www.mongodb.com/try/download/community
- OU MongoDB Compass (version graphique, plus simple)

---

## ETAPE 2 : TELECHARGE LE CODE

1. Telecharge le code depuis Emergent ou GitHub
2. Tu auras un fichier .zip
3. Extrait le fichier .zip dans un dossier simple : C:\DarkNexus\

---

## ETAPE 3 : LANCE L'INSTALLATION

1. Va dans le dossier : C:\DarkNexus\[nom-du-dossier]\
2. Cherche le fichier : INSTALLATION_AUTOMATIQUE_v2.bat
3. Double-clic dessus !

Le script va :
- Verifier Python et Node.js
- Installer Yarn automatiquement
- Creer les fichiers .env
- Installer toutes les dependances (yarn install, pip install)
- Configurer ta cle OpenAI
- Creer le lanceur de l'app

Duree : 5-10 minutes
Laisse-le faire tout seul !

---

## ETAPE 4 : LANCE L'APPLICATION

1. Assure-toi que MongoDB est lance (ouvre MongoDB Compass)
2. Double-clic sur : LANCER_DARK_NEXUS.bat
3. Deux fenetres vont s'ouvrir (Backend et Frontend)
4. Ne les ferme pas !
5. Ton navigateur va s'ouvrir sur http://localhost:3000

## EN CAS DE PROBLEME

### Python n'est pas reconnu
Solution : Reinstalle Python en cochant "Add Python to PATH"

### Node n'est pas reconnu
Solution : Installe Node.js depuis https://nodejs.org/

### Le backend ne demarre pas
Solution : Lance MongoDB (MongoDB Compass ou mongod)

### yarn install echoue
Solution : 
1. Ouvre CMD en tant qu'administrateur
2. Va dans le dossier frontend : cd C:\DarkNexus\...\frontend
3. Tape : npm install -g yarn
4. Relance le script d'installation
