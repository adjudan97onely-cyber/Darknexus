# Chef IA - Version mobile sans PC

Objectif: avoir l'application sur telephone sans dependre du PC allume.

## Option recommandee: PWA installee depuis Vercel

1. Deployer le frontend sur Vercel.
2. Ouvrir l'URL sur le mobile.
3. Installer sur l'ecran d'accueil (Android ou iOS).

## Option Android APK

1. Utiliser l'URL de production.
2. Generer une app Android via https://www.pwabuilder.com/
3. Installer l'APK sur le telephone.

## Note importante

Un ZIP GitHub n'est pas une application mobile installable.
Pour une vraie installation mobile, il faut une PWA installee ou un APK.
