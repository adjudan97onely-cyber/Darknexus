  🎯 ÉTAPE 1 : LIS LE GUIDE COMPLET
═══════════════════════════════════════════════════════════════════

📄 Ouvre ce fichier avec le Bloc-notes :
   👉 GUIDE_INSTALLATION_SIMPLE.md

🎯 Ce guide contient :
   ✅ Liste des pré-requis (Python, Node.js, MongoDB)
   ✅ Instructions d'installation détaillées
   ✅ Comment lancer l'application
   ✅ Solutions aux problèmes courants
   ✅ Astuces et raccourcis

═══════════════════════════════════════════════════════════════════
  🚀 ÉTAPE 2 : LANCE L'INSTALLATION (1 SEULE FOIS)
═══════════════════════════════════════════════════════════════════

📄 Double-clic sur ce fichier :
   👉 INSTALLATION_AUTOMATIQUE.bat

🎯 Ce script va :
   ✅ Vérifier que Python et Node.js sont installés
   ✅ Créer les fichiers .env automatiquement (c'est ça qui te bloquait !)
   ✅ Installer toutes les dépendances
   ✅ Configurer ta clé OpenAI
   ✅ Créer le lanceur de l'app

⏳ Durée : 3-5 minutes
☕ Laisse-le faire, il gère tout seul !

═══════════════════════════════════════════════════════════════════
  🎮 ÉTAPE 3 : LANCE TON APPLICATION (À CHAQUE UTILISATION)
═══════════════════════════════════════════════════════════════════

⚠️  AVANT DE LANCER : Assure-toi que MongoDB est lancé !
   (Ouvre MongoDB Compass ou lance 'mongod' en ligne de commande)

📄 Double-clic sur ce fichier :
   👉 LANCER_DARK_NEXUS.bat

🎯 Ce script va :
   ✅ Démarrer le backend (moteur de l'app)
   ✅ Démarrer le frontend (interface)
   ✅ Ouvrir ton navigateur automatiquement

⚠️  NE FERME PAS les 2 fenêtres qui s'ouvrent !
   (Sinon l'app s'arrête)
